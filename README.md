🎧 Spotify Clone

Clone do Spotify desenvolvido para praticar desenvolvimento web e integração entre front-end e back-end.

🚀 Tecnologias utilizadas
• HTML5
• CSS3
• JavaScript
• API própria (conversão de MP3 para hexadecimal)

⚙️ Funcionalidades
• Interface inspirada no Spotify
• Player de música
• Navegação entre playlists
• Conversão de arquivos MP3 para representação hexadecimal

🧠 Objetivo do projeto
Praticar manipulação do DOM, responsividade, integração com API e organização de código.

📌 Status
Projeto desenvolvido para fins de estudo e aprimoramento em Desenvolvimento de Sistemas.
