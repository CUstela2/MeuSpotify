🎧 Spotify Clone

Clone do Spotify desenvolvido para praticar desenvolvimento web e integração entre front-end e back-end.

🚀 Tecnologias utilizadas
	•	HTML5
	•	CSS3
	•	JavaScript
	•	API própria (Speech-to-Text e Text-to-Speech)

⚙️ Funcionalidades
	•	Interface inspirada no Spotify
	•	Player de música
	•	Navegação entre playlists
	•	Conversão de MP3 em texto
	•	Leitura automática do texto gerado

🧠 Objetivo do projeto

Praticar manipulação do DOM, responsividade, integração com API e organização de código.

📌 Status

Projeto para fins de estudo e aprimoramento em Desenvolvimento de Sistemas
